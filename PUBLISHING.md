# GitHub Pages

To publish the Prompt Sharding landing page:

1. Go to **Settings → Pages** in this repo.
2. Set the source to **Deploy from a branch**.
3. Choose the **main** branch.
4. Choose the **/(root)** folder.
5. Save, then open the published URL.

If you want this page to be the site root, rename `landing.html` to `index.html`.

## Before launch

- Replace the placeholder Tally URL in `landing.html`
- Add your real waitlist form
- Optionally add an Open Graph image and favicon
